# 连接控件和统计控件

## 连接控件

### 合并记录

> 合并记录就是将旧数据和新数据进行合并比较。**合并记录前需先使用排序记录**

比较的结果有以下可能：

- "identical"  —  旧数据和新数据一样
- "changed"  —   数据发生了变化
- "new"   —  新数据中有，旧数据中没有
- "deleted"  —  新数据中没有，旧数据中有
